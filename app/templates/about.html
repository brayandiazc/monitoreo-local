Acerca del Proyecto

Sistema de Monitoreo Local

Este proyecto demuestra un entorno completo de monitoreo local utilizando Docker Compose. Incluye todas las herramientas necesarias para monitorear aplicaciones en tiempo real.

Componentes del Sistema
  • Flask App: Aplicación web con logging y métricas
  • Prometheus: Recolección y almacenamiento de métricas
  • Grafana: Visualización de métricas y dashboards
  • Elasticsearch: Motor de búsqueda y análisis
  • Logstash: Procesamiento de logs
  • Kibana: Visualización de logs
Características
  • Logging estructurado en JSON
  • Métricas automáticas de Flask
  • Dashboards preconfigurados
  • Visualización de logs en tiempo real
  • Alertas configurables
  • Entorno completamente containerizado
Información Técnica
Puertos Utilizados:
  • 5000: Aplicación Flask
  • 9090: Prometheus
  • 3000: Grafana
  • 5601: Kibana
  • 9200: Elasticsearch
  • 5044: Logstash
Tecnologías:
  • Python 3.9+
  • Flask 2.3.3
  • Docker & Docker Compose
  • Prometheus
  • Grafana
  • ELK Stack
